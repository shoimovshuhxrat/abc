1-dars | Kirish

Assalomu alaykum va xush kelibsiz! Ushbu darsda biz HTML dasturlash tiliga kirish qilamiz va uning asosiy tushunchalarini o'rganamiz.

HTML (HyperText Markup Language) - bu veb-sahifalarni yaratish uchun ishlatiladigan standart belgilash tili. HTML veb-sahifaning tuzilishini belgilaydi va brauzerga qanday ko'rsatish kerakligini aytadi.

HTML hujjati bir qator elementlardan iborat bo'lib, ular brauzerga kontentni qanday ko'rsatish kerakligini aytadi. HTML elementlari teglar bilan belgilanadi.

<!DOCTYPE html>
<html>
  <head>
    <title>Mening birinchi veb-sahifam</title>
  </head>
  <body>
    <h1>Salom, dunyo!</h1>
    <p>Bu mening birinchi veb-sahifam.</p>
  </body>
</html>

Keyingi darslarda biz HTML elementlari va ularning atributlari haqida batafsilroq o'rganamiz.

2-dars | HTML haqida

HTML (HyperText Markup Language) veb-sahifalarni yaratish uchun ishlatiladigan standart belgilash tilidir. U veb-sahifaning tuzilishini belgilaydi.

HTML elementlari ochilish va yopilish teglari bilan belgilanadi. Masalan, <p> va </p> teglari paragrafni belgilaydi.

HTML hujjati quyidagi asosiy qismlardan iborat:

  • <!DOCTYPE html> - hujjat turi deklaratsiyasi
  • <html> - HTML hujjatining ildiz elementi
  • <head> - meta-ma'lumotlar, sarlavha va boshqa resurslarni o'z ichiga oladi
  • <title> - brauzer sarlavhasida ko'rsatiladigan sahifa sarlavhasi
  • <body> - sahifaning asosiy mazmuni
<!DOCTYPE html>
<html lang="uz">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML haqida</title>
  </head>
  <body>
    <h1>HTML haqida</h1>
    <p>HTML veb-sahifalarni yaratish uchun ishlatiladigan belgilash tili.</p>
  </body>
</html>

3-dars | Hujjatni formatlash

HTML-da matnni formatlash uchun turli xil teglar mavjud. Ular matnning ko'rinishini o'zgartirish va uni tuzilishli qilish uchun ishlatiladi.

Sarlavha teglari: HTML-da 6 ta sarlavha tegi mavjud - h1 dan h6 gacha. h1 eng katta sarlavha, h6 esa eng kichik.

<h1>Bu h1 sarlavha</h1>
<h2>Bu h2 sarlavha</h2>
<h3>Bu h3 sarlavha</h3>
<h4>Bu h4 sarlavha</h4>
<h5>Bu h5 sarlavha</h5>
<h6>Bu h6 sarlavha</h6>

Paragraf tegi: Matnni paragraf sifatida belgilash uchun <p> tegi ishlatiladi.

<p>Bu bir paragraf.</p>
<p>Bu boshqa paragraf.</p>

Matn formatlash teglari: Matnni qalin, kursiv yoki tagiga chizilgan qilish uchun quyidagi teglar ishlatiladi:

<b>Qalin matn</b> yoki <strong>Muhim matn</strong>
<i>Kursiv matn</i> yoki <em>Ta'kidlangan matn</em>
<u>Tagiga chizilgan matn</u>
<mark>Belgilangan matn</mark>
<small>Kichik matn</small>
<del>O'chirilgan matn</del>
<ins>Kiritilgan matn</ins>
<sub>Pastki indeks</sub>
<sup>Yuqori indeks</sup>

4-dars | Taglar haqida batafsi

HTML-da turli xil maqsadlar uchun ko'plab teglar mavjud. Bu darsda biz eng ko'p ishlatiladigan teglarni ko'rib chiqamiz.

Ro'yxat teglari: HTML-da tartibli va tartibsiz ro'yxatlarni yaratish uchun teglar mavjud.

<!-- Tartibsiz ro'yxat -->
<ul>
  <li>Olma</li>
  <li>Banan</li>
  <li>Apelsin</li>
</ul>

<!-- Tartibli ro'yxat -->
<ol>
  <li>Birinchi qadam</li>
  <li>Ikkinchi qadam</li>
  <li>Uchinchi qadam</li>
</ol>

Havola tegi: Boshqa sahifalarga yoki resurslarga havolalar yaratish uchun <a> tegi ishlatiladi.

<a href="https://www.example.com">Bu yerga bosing</a>
<a href="about.html">Biz haqimizda</a>
<a href="mailto:info@example.com">Bizga xabar yuboring</a>
<a href="tel:+998901234567">Qo'ng'iroq qiling</a>

Rasm tegi: Rasmlarni ko'rsatish uchun <img> tegi ishlatiladi.

<img src="rasm.jpg" alt="Rasm tavsifi" width="300" height="200">

Jadval teglari: Jadvallarni yaratish uchun quyidagi teglar ishlatiladi:

<table>
  <tr>
    <th>Ism</th>
    <th>Familiya</th>
    <th>Yosh</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>Valiyev</td>
    <td>25</td>
  </tr>
</table>

5-dars | Attributlar haqida batafsi

HTML elementlari qo'shimcha ma'lumotlarni taqdim etish uchun attributlarga ega bo'lishi mumkin. Attributlar ochilish tegida ko'rsatiladi va odatda nom-qiymat juftligi sifatida yoziladi.

Umumiy attributlar: Ko'pchilik HTML elementlari uchun umumiy bo'lgan attributlar:

  • id - elementga noyob identifikator beradi
  • class - elementga bir yoki bir nechta sinf nomlarini beradi
  • style - elementga inline CSS uslublarini qo'llaydi
  • title - element haqida qo'shimcha ma'lumot beradi (ko'pincha tooltip sifatida ko'rsatiladi)
<p id="intro" class="text-large important" style="color: blue;" title="Kirish paragrafi">
  Bu paragraf turli xil attributlarga ega.
</p>

Havola attributlari: <a> tegi uchun maxsus attributlar:

<a href="https://www.example.com" target="_blank" rel="noopener noreferrer">Yangi oynada ochish</a>

Rasm attributlari: <img> tegi uchun maxsus attributlar:

<img src="rasm.jpg" alt="Rasm tavsifi" width="300" height="200" loading="lazy">

Form elementlari attributlari: Form elementlari uchun maxsus attributlar:

<input type="text" name="username" placeholder="Foydalanuvchi nomi" required>
<input type="password" name="password" placeholder="Parol" minlength="8" required>
<input type="email" name="email" placeholder="Email manzil" autocomplete="email">
<input type="checkbox" name="agree" checked> Shartlarga roziman
<button type="submit" disabled>Yuborish</button>
